# ai_voice_bot
AI Voice Bot using SarvamAI is a Python-based conversational assistant that converts user speech or text into AI-generated responses and returns them as natural voice output. It integrates speech-to-text, LLM processing, and text-to-speech to create a simple end-to-end voice interaction system with modular and maintainable code structure.

# AI Voice Bot using SarvamAI

This project implements a conversational AI voice bot using the SarvamAI API.

## Features

- Speech to Text
- LLM conversation
- Text to Speech
- Conversation memory
- Modular Python architecture

## Tech Stack

Python  
SarvamAI  
SoundDevice  
NumPy
